# PANDA

Official repository for the paper:

**PANDA: Progressive Adaptive Noise Denoising Architecture for Graph-Based Multimodal Fake News Detection**

This repository will host the implementation, experimental scripts, and processed configuration files for PANDA.

## Status

The code is being organized and will be released upon publication or acceptance.

## Overview

PANDA is a three-stage framework for mitigating noisy LLM pseudo-labels in graph-based multimodal fake news detection:

1. Distributional Confidence Estimation
2. Zone of Proximal Development Selection
3. Prototype-guided Contrastive Rectification
